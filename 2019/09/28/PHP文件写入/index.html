利用获取的PHP权限写shell

在获取到一定的PHP权限的时候可以执行一些函数、利用使用system调用系统命令，从而拿到shell；在可以执行命令的时候怎么拿到shell呢，以往出现类似的RCE的时候，都是利用别人写好的工具，而且大多数的EXP都只是利用漏洞执行系统命令验证，利用的时候要拿到交互式的shell才可以。这种情况可以选择反弹shell，但是有个人习惯且是个网站，反弹的shell功能有限，这里我选择可以利用的系统命令或者PHP的文件操作函数进行一句话的写入，或者写入冰蝎马用客户端连接，这样更方便操作。（ps:有时候冰蝎马由于语法无法写入，普通一句话特征又太明显，可以通过普通一句话上传冰蝎马，解决菜刀年久失修的问题）
